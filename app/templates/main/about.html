Under the Hood

About This Site

This portfolio is itself a project — designed, built, and self-hosted from scratch on a single-board computer running in a home lab. No cloud, no managed hosting, no templates.

Hardware

Application Server

Libre Computer AML-S905X-CC

"Le Potato" — ARM Cortex-A53 · Debian 12 Bookworm

  • Runs the Flask application and Gunicorn WSGI server
  • Communicates via Unix socket to Nginx reverse proxy
  • Development via VSCode SSH Remote from any machine

Network Gateway

Raspberry Pi 4 Live

ARM Cortex-A72 · Debian Trixie · "Julie"

  • Runs cloudflared — outbound Cloudflare Tunnel, no open ports on the router
  • Runs Pi-hole for network-level DNS filtering
  • ASUS RT-AX86U router (AX5700) with AiProtection

Software Stack

Layer Technology Role
Framework Flask Python web framework — blueprints, Jinja2 templates, SQLAlchemy ORM
Database SQLite Stores resume / work history data; queried live on each page load
CSS Tailwind CSS v4 Utility-first CSS; compiled by a standalone CLI watcher running as a systemd service
WSGI Gunicorn Serves the Flask app via Unix socket; managed by systemd
Reverse Proxy Nginx Handles static file serving and proxies requests to Gunicorn
Tunnel Cloudflare Tunnel Outbound-only tunnel from the RPi4 to Cloudflare's edge — no ports exposed to the internet, home IP never revealed
Visualizations HTML5 Canvas / JS Interactive Lab visualizations rendered entirely in the browser — physics computed client-side, no server load

Why Self-Hosted?

  • The site is a demonstration of software and systems skills — deploying it on managed infrastructure would miss the point.
  • Running on a $15 ARM SBC with no cloud costs is a practical constraint that informs every technical decision — from choosing SQLite over Postgres to keeping Lab visualizations client-side.
  • The Cloudflare Tunnel approach demonstrates that secure public hosting from a home network doesn't require exposed ports, a static IP, or a monthly bill.
