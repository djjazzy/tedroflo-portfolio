BLDC Motor Control System

A complete closed-loop Field Oriented Control (FOC) system for a brushless DC motor — designed, built, and programmed by one engineer across hardware, firmware, and mechanical disciplines. Demonstrates sensorless FOC, sensored FOC, speed control, and position control using a TI C2000 real-time microcontroller.

Hardware Stack

Component Part Role
MCU TI LAUNCHXL-F280049C C2000 Piccolo — real-time control DSP with ePWM, CLA, ADC, eQEP
Gate Driver TI BOOSTXL-DRV8323RS 3-phase gate driver with integrated current shunt amplifiers (SPI config)
Motor TSINY JK42BLS01 NEMA 17 frame BLDC — 3-phase, no encoder (sensorless to start)
Encoder AS5047D Pending 14-bit magnetic absolute encoder — SPI/ABI interface for sensored FOC

Project Phases

Phase 1 — Sensorless FOC Speed Control

In Progress
  • Back-EMF / FAST observer sensorless algorithm (TI MotorControl SDK)
  • Open-loop startup sequence → closed-loop handoff
  • PI speed controller tuned via CCS real-time watch/graph
  • Phase current reconstruction from DRV8323RS inline shunt amps

Phase 2 — Sensored FOC (Hall → AS5047D)

Pending
  • Step 2a — use onboard Honeywell Hall sensors (120° electrical) for sensored trapezoidal commutation; no additional hardware required
  • Step 2b — add AS5047D magnetic encoder (14-bit, SPI) for high-resolution rotor angle; 3D print precision encoder housing in Fusion 360
  • Upgrade FOC from estimated (FAST observer) to measured rotor angle
  • Improved low-speed torque and stable operation down to near-zero RPM

Phase 3 — Position Control

Pending
  • Outer position loop cascaded over FOC speed loop
  • Trapezoidal velocity profiling for smooth point-to-point moves
  • Load disturbance testing with 3D printed flywheel

Phase 4 — Custom PCB Design & Fabrication

Pending

  • Full schematic capture — MCU, gate driver, current sensing, power stage, encoder interface, connectors
  • PCB layout with proper power plane separation, current loop minimization, and thermal considerations for the gate driver
  • Generate and submit gerbers for board fabrication
  • Publish KiCad project, gerbers, schematic PDF, and layout to portfolio
